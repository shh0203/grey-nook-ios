# assets/images/

This directory holds static image assets for Grey Nook.

In the first cycle the only declared asset is a placeholder path used
by the mock chat (`assets/images/placeholder_flower.jpg`). The
`ChatBubble` for image messages renders an on-brand gradient card
instead, so the file does not need to be shipped yet.

When real assets land, drop them here and reference them via
`pubspec.yaml`'s `flutter.assets:` list.
